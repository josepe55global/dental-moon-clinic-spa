# Instrucciones para abrir y modificar en Antigravity

## 1. Carpeta que debes abrir
Abre en Antigravity la carpeta completa:

`dental-moon-antigravity-ready`

No abras solo `index.html`. Debes abrir la carpeta completa porque el sitio depende de:

- `css/styles.css`
- `js/main.js`
- `assets/`
- imágenes, logo, favicon y archivos auxiliares

## 2. Qué archivo editar principalmente
Los archivos principales son:

- `index.html` para estructura y contenido.
- `css/styles.css` para diseño, responsive, colores, animaciones y mobile.
- `js/main.js` para WhatsApp, menú, guía de consulta y formularios.

## 3. Imágenes nuevas autorizadas
Las imágenes nuevas están en:

`assets/images/nuevas-autorizadas/`

Uso sugerido:

- `01-hero-revision-dental.png`: imagen principal del hero.
- `02-agenda-consulta.png`: CTA de agendar consulta o bloque comercial.
- `03-equipo-clinica.png`: galería o sección de confianza.
- `04-ortodoncia-agenda.png`: servicio de ortodoncia/alineadores.
- `05-protesis-dental.png`: servicio de prótesis dentales.
- `06-fachada-clinica.png`: ubicación/fachada.
- `07-atencion-clinica.png`: galería o revisión clínica.

## 4. Prompt para Antigravity
Copia y pega el contenido completo de:

`01_PROMPT_ANTIGRAVITY.md`

Después dile a Antigravity:

“Aplica estos cambios directamente sobre esta carpeta. Revisa index.html, css/styles.css y js/main.js. Conserva estructura estática lista para GitHub y Vercel.”

## 5. Qué pedirle después de aplicar cambios
Cuando termine, pídele:

- Revisar versión móvil.
- Verificar que WhatsApp abra con mensaje preparado.
- Verificar que no haya datos pendientes excepto correo, Google Maps y aviso de privacidad.
- Verificar que las fotos estén optimizadas.
- Crear capturas de escritorio y móvil si es posible.
